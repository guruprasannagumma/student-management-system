# Student Management System

## 🚀 Project Overview
A full-stack web application to manage student records with complete CRUD operations.

## 🛠️ Technologies Used
- Java (Spring Boot)
- Oracle Database
- HTML, CSS, JavaScript

## ✨ Features
- Add Student
- View Students
- Update Student
- Delete Student

## 📌 How to Run
1. Run Spring Boot application
2. Open browser:
   http://localhost:8082

## 📸 Output
- Displays student list
- Add/Edit/Delete functionality
